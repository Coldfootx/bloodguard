This is the Vanilla Version (on GitHub). Derivatives must be clearly marked as such.

Download: Windows - Linux - Mac (Mac not tested)

On Windows, extract the archive and run the .exe to launch it.
On macOS, extract the archive and double click on the app to launch it.
On Linux, give the AppImage executable rights and run "sudo apt-get install libfuse2" and run "./*.AppImage --no-sandbox"
libfuse2 and --no-sandbox might be optional

When launching the game on Windows and macOS, you might get a notice about security. Choose to run the game anyway.
On macOS, you may need to open Security and Privacy tab in System Preferences.
